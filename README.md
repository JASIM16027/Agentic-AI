## 1. Theory: Understanding Agentic AI (তত্ত্ব: Agentic AI বোঝা)

### 1.1 Definition of Agentic AI (Agentic AI-এর সংজ্ঞা)
- **Agentic AI** হলো এমন একটি AI সিস্টেম যা **autonomy** (স্বায়ত্তশাসন), **reasoning** (যুক্তি), এবং **action-taking** ক্ষমতা ব্যবহার করে নির্দিষ্ট লক্ষ্য অর্জন করে, ন্যূনতম মানুষের হস্তক্ষেপে।  
- সাধারণ AI (যেমন, chatbot) শুধুমাত্র প্রশ্নের উত্তর দেয়, কিন্তু Agentic AI স্বাধীনভাবে কাজ সম্পাদন করে।  
- **উদাহরণ**: একটি Agentic AI ওয়েব থেকে তথ্য সংগ্রহ করতে পারে, সংক্ষিপ্ত করতে পারে, এবং ইমেইলের মাধ্যমে রিপোর্ট পাঠাতে পারে—একটি একক লক্ষ্যের ভিত্তিতে।  

### 1.2 Key Characteristics (প্রধান বৈশিষ্ট্য)
Agentic AI-এর চারটি মূল বৈশিষ্ট্য:  
- **Autonomy (স্বায়ত্তশাসন)**: মানুষের নির্দেশ ছাড়াই কাজ করে।  
- **Reasoning (যুক্তি)**: ReAct, Chain-of-Thought (CoT), বা Tree-of-Thought (ToT) পদ্ধতি ব্যবহার করে জটিল সমস্যা সমাধান করে।  
- **Tool Use (টুল ব্যবহার)**: API, ডাটাবেস, বা ব্রাউজারের মতো বাহ্যিক সিস্টেমের সাথে সংযোগ স্থাপন করে।  
- **Memory (মেমোরি)**: Short-term (ইন-মেমোরি) বা long-term (vector database) কনটেক্সট ধরে রাখে।  

### 1.3 Evolution from LLMs to Agentic AI (LLM থেকে Agentic AI-এর বিবর্তন)
Agentic AI, Large Language Models (LLM)-এর উপর ভিত্তি করে তৈরি, কিন্তু এটি LLM-এর সীমাবদ্ধতা অতিক্রম করে। বিবর্তনের ধাপগুলো:  
- **Pre-LLM Era (2017-এর আগে)**:  
  - Rule-based systems এবং early machine learning models ছিল।  
  - এগুলো টাস্ক-নির্দিষ্ট এবং সাধারণ বুদ্ধিমত্তার অভাব ছিল (যেমন, expert systems)।  
- **LLM Era (2017–2022)**:  
  - BERT, GPT-3-এর মতো মডেল natural language understanding এবং generation-এ বিপ্লব এনেছে।  
  - সীমাবদ্ধতা: প্রতিটি কাজের জন্য মানুষের প্রম্পট প্রয়োজন।  
- **Agentic AI Era (2022–বর্তমান)**:  
  - ফ্রেমওয়ার্ক যেমন LangChain, AutoGen, CrewAI LLM-এর ক্ষমতা বাড়িয়েছে:  
    - **Tool Integration**: API, web scraping, ডাটাবেসের সাথে সংযোগ।  
    - **Memory**: Vector databases (যেমন Pinecone, FAISS) দিয়ে কনটেক্সট ধরে রাখা।  
    - **Reasoning Frameworks**: ReAct, CoT, ToT।  
    - **Multi-Agent Collaboration**: এজেন্টদের টিম তৈরি।  
  - **উদাহরণ**: LLM শুধু “আবহাওয়া কেমন?” উত্তর দিতে পারে, কিন্তু Agentic AI API থেকে আবহাওয়ার ডেটা সংগ্রহ করে, সংক্ষিপ্ত করে, এবং বৃষ্টির পূর্বাভাস থাকলে রিমাইন্ডার সেট করে।  

### 1.4 Why This Theory Matters (কেন তত্ত্ব গুরুত্বপূর্ণ)?
- Agentic AI-এর সংজ্ঞা এবং বিবর্তন বোঝা আপনাকে এর কাজের প্রক্রিয়া এবং সম্ভাবনা বুঝতে সাহায্য করে।  
- এটি static LLM থেকে dynamic, goal-driven agents-এর দিকে যাওয়ার পথ দেখায়, যা আপনি পরবর্তীতে LangChain এবং AutoGen দিয়ে বাস্তবায়ন করবেন।  
- এই ধারণাগুলো পরবর্তী দিনগুলোর জন্য ভিত্তি তৈরি করে, যেমন reasoning techniques এবং tool integration।

## 2. Coding: Install LangChain and AutoGen; Write a Basic LLM Query

### 2.1 Objective (উদ্দেশ্য)
- Development environment সেট আপ করুন।  
- LangChain এবং AutoGen ব্যবহার করে একটি basic LLM query তৈরি করুন (যেমন, “What is Agentic AI?” প্রশ্ন করা)।  
- এটি ফ্রেমওয়ার্কের সাথে পরিচিতি এবং সেটআপ যাচাই করবে।  

### 2.2 Steps (ধাপগুলো)
1. **Install LangChain and AutoGen (LangChain এবং AutoGen ইনস্টল করুন)**:  
   - **LangChain**: Python ফ্রেমওয়ার্ক যা LLM-এর সাথে tools, memory, এবং workflows ইন্টিগ্রেট করে।  
   - **AutoGen**: Multi-agent systems তৈরির জন্য, যেখানে এজেন্টরা সহযোগিতা করে।  
   - টার্মিনালে কমান্ড:  
     ```bash
     pip install langchain langchain-openai pyautogen
     ```  
   - Python 3.8+ এনভায়রনমেন্টে ইনস্টল করুন।  

2. **Set Up API Keys (API কী সেট আপ করুন)**:  
   - OpenAI API কী প্রয়োজন (https://platform.openai.com/ থেকে পান)।  
   - Environment variable হিসেবে সেট করুন অথবা কোডে হার্ডকোড করুন (সুরক্ষার জন্য env variable প্রস্তাবিত)।  
   - Optional: Hugging Face-এর local models পরে ব্যবহার করতে পারেন।  

3. **Write a Basic LLM Query (Basic LLM কোয়েরি লিখুন)**:  
   - একটি Python স্ক্রিপ্ট তৈরি করুন যা LLM-কে প্রম্পট পাঠায় এবং উত্তর প্রিন্ট করে।  
   - প্রম্পট: “What is Agentic AI?”  

### 2.3 Example Code

#### **LangChain with OpenAI**
```python
# ইনস্টল করুন: pip install langchain langchain-openai
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate

# OpenAI API কী সেট করুন
import os
os.environ["OPENAI_API_KEY"] = "your-openai-api-key"  # আপনার কী দিয়ে প্রতিস্থাপন করুন

# LLM ইনিশিয়ালাইজ করুন
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0.7)

# প্রম্পট তৈরি করুন
prompt = ChatPromptTemplate.from_template("Explain what Agentic AI is in 50 words.")

# LLM কোয়েরি করুন
response = llm.invoke(prompt.format())
print(response.content)
```

#### **AutoGen with OpenAI**
```python
# ইনস্টল করুন: pip install pyautogen
from autogen import AssistantAgent, UserProxyAgent, config_list_from_json

# OpenAI API কী কনফিগার করুন
config_list = [{"model": "gpt-3.5-turbo", "api_key": "your-openai-api-key"}]

# এজেন্ট ইনিশিয়ালাইজ করুন
assistant = AssistantAgent(name="Assistant", llm_config={"config_list": config_list})
user_proxy = UserProxyAgent(name="User", human_input_mode="NEVER", max_consecutive_auto_reply=1)

# কথোপকথন শুরু করুন
user_proxy.initiate_chat(assistant, message="What is Agentic AI?")
```
